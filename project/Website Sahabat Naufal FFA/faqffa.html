Clear Cache

Jika masih belum paham silahkan buka tiket untuk dibantu


Q : Kak Untuk Hapus Cache Bagaimana ya kak?

A : Langkah- Langkah Hapus Cache

- Klik kanan Pada Aplikasi FiveM kemudian pilih open file location

- Lalu masuk kedalam folder FiveM Application Data

- Setelah itu masuk ke folder data

- Hapus semua folder kecuali folder game-storage



Extended Texture Budget


Cara ini digunakan ketika kamu dapat masalah rendering (missing texture/bubur) Semakin kecil penggunaan maka penggunakan VGA semakin ringan

Q : kak kok saya jelly/bubur saat masuk server ya kak?

A : kalau ada masalah seperti ini kamu bisa melakukan hal

Esc -> Setting -> Graphic -> Extended Texture Budget



Steam tidak terdeteksi


Q : Min kok saya tiba tiba di kick ya min baru masuk server dengan alasan "Akun Kamu telah terdeteksi, pastikan steam dan rockstar sudah berjalan

A : Ketika kamu mempunyai masalah seperti ini kamu bisa melakukan

- Close FiveM kamu

- Login akun steam kamu

- Dan masuk server kembali



Stuck Updating Game Cache


Q : Min saya kan updating game cache sudah 100% tapi stuck, solusinya bagaimana ya kak?

A : Ketika kamu mempunyai masalah seperti diatas kamu harus melakukan verify game Grand Theft Auto 5 kamu

>


Verify GTA 5 Epic Games

- Buka Epic games kamu

- Buka Library dan cari game grand theft auto 5

- Klik titik 3 pada games kamu

- Lalu klik Verify

- Verfy akan diproses tunggu sampai verify selesai

- Lalu buka FiveM kembali dan masuk ke server



Verify GTA 5 Steam

- Buka steam kamu

- Buka Library dan cari game grand theft auto 5

- Klik kanan pada gamemu lalu klik properties

- nanti akan muncul pop up dan ada beberapa pilihan, lalu kamu klik Local Files

- Lalu klik Verify Integrity of games files

- Verfy akan diproses tunggu sampai verify selesai

- Lalu buka FiveM kembali dan masuk ke server



Progres Downloading Assets

Q : min cara menunjukkan proses download ketika in-game bagaimana ya?

A : Ketika kamu ingin menampilkan proses download ketika in-game yaitu :

- Buka FiveM kamu

- Buka menu settings dipojok kanan atas

- Centang "Display progress when downloading in game

- VSetelah itu masuk server dan akan muncul di pojok kanan atas



Kamu masih terdeteksi didalam

Q : Kak kok aku tidak bisa masuk server ya dengan alasan "Kamu masih terdeteksi didalam, harap tunggu beberpaa saat dan mencoba lagi

A : Untuk kamu yang punya masalah seperti itu dikarenakan steam kamu masih nyangkut di server jadi untuk solusi hal ini :

- Tutup FiveM

- Tunggu sekitar 1 - 5 menit

- Buka FiveM kembali

- Masuk ke server kembali



Tenggelem Dibase

Q : min cara solusi untuk tenggelem dibase dan gak bisa liat dibase bagaimana ya min?

A : Untuk kamu yang punya masalah seperti ini kamu perlu :

- Gunakan First Person atau pandangan pertama

- Jalan ke menu teleport

- Teleport ke suatu tempat

- Balik ke base awal



Mengetahui Steam Hex

Q : min cara mengetahui steamhex kita sendiri bagaimana ya?

A : untuk kamu yang ingin mengetahui steamhex kamu sendiri kamu bisa melakukan :

- Kamu bisa klik link ini (http://vacbanned.com/engine/check)

- Setelah kamu buka link tersebut maka akan terbuka menu dari website itu sendiri

- Setelah itu kamu bisa cari nama steamkamu/ steam profile kamu dihalaman search di pojok kanan atas

- setelah itu kamu akan menemukan hex kamu sesuai digambar
